# FlatMatch | Premium Flatmate & Rent Finder

A modern, AI-powered platform for finding compatible flatmates and the perfect room. Built with Next.js, Node.js, and PostgreSQL.

🚀 **Live Website:** [https://rent-flatmate-finder.vercel.app/](https://rent-flatmate-finder.vercel.app/)

## Features
- **AI Compatibility Matching**: Find flatmates who match your lifestyle.
- **Real-Time Chat**: Secure messaging using Socket.io.
- **Beautiful UI**: Modern dark-mode interface.
- **Secure Authentication**: JWT-based login and registration.
